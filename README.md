# github_tags
Projeto para criar/listar tags de um determinado repositório.

## Como rodar o script
Clonar o projeto;
Criar um arquivo .env na raíz do diretório, seguindo as informações do .env.example;
Instalar o curl na máquina linux
`sudo apt update`
`sudo apt upgrade`
`sudo apt install curl`
Executar o script
`bash script.sh`
